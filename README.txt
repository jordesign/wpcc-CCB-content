== Description ==
This plugin is an addon for WP Church Center. It allows user to sync Group and Event datat with Church Community Builder and then display the content in cards.

== Installation ==
1). Download the plugin

2). Install the plugin by either:
extracting the .zip file and uploading the entire wp-church-center folder to the /wp-content/plugins/ directory of our web server.
go to Plugins > Add New in the WordPress Admin and upload the .zip file.
(You'll need at least v1.2.7 of WP Church Center to use this Addon).

3). Activate the Plugin. You should now be able to select 'CCB Content' as a card type.

4). Select the type of CCB content you'd like and fill out the additional details.
